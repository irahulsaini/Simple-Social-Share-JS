Installation:

Put these two codes before </body>
Please note: if you have installed jQuery then no need to include jQuery again

<!-- jQuery or jQuery Slim Library Required -->
<script src="https://code.jquery.com/jquery-3.4.1.min.js" crossorigin="anonymous" type="text/javascript" ></script>

<!-- Social Share JS -->
<script src="./path/to/socialshare.js" type="text/javascript" ></script>

You can also integrate social share js source code into your site's main javascript file to avoid network request.

Examples:

Simple Social Share Plugin is very simple and easy to integrate. It doesn't harm your any design or development. To enable sharing you'll need to simply add data-share attribute to element. Add data-share="network_name" into your button or link and it will automatically integrate social share system

<button data-share="facebook">Share on Facebook</button>
<button data-share="twitter">Share on Twitter</button>
<button data-share="linkedin">Share on Linkedin</button>
<button data-share="pinterest">Share on Pinterest</button>

Social Network Attributes:

Name Code Details
Facebook data-share="facebook" No extra tag/attributes required
Twitter data-share="twitter" No extra tag/attributes required
Pinterest data-share="pinterest" It require og:image tag or data-image to share image on pinterest
Linkedin data-share="linkedin" No extra tag/attributes required
Whatsapp data-share="whatsapp" User must logged in Whatsapp
Telegram data-share="telegram" User must logged in Telegram
Blogger data-share="blogger" Create a Post containing page information
Gmail data-share="gmail" Compose a New Email (Login Required)
Mail data-share="mail" User need to configure app into their device to share via mail
Optional Attributes:
Page Title data-title="Page Title" Custom Page Title if you don't want to display default page title in social sharing
Page Meta Description data-description="Page Description" Custom Page description if you don't want to display default page title in social sharing
Page URL/Canonical data-url="http://www.yoursite.com/pagename/" Custom Page URL if you don't want to display default page title in social sharing
OG Image data-image="http://www.yoursite.com/myimage.jpg" Custom Image URL for Open Graph Image if you don't want to display default page title in social sharing
